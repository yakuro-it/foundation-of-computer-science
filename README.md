# ST4015CMD: Foundation of Computer Science

## Content in this Respository:
This repository contains scripts, SQL files, and diagrams supporting:
- **Task 1:** Encoding formats and protocol integration
- **Task 2:** Classroom seating arrangement (P vs NP, brute force, heuristic)
- **Task 3:** Database normalization, ERD, and SQL queries

---

## Github Clone Command
```bash
git clone https://github.com/yakuro-it/foundation-of-computer-science.git
cd foundation-of-computer-science
```

```bash
cd task1-simulation
python3 task1_encoding_protocol_demo.py
```
